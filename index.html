Les Cheat-Sheet à la Guytou

tab complemente la synthaxe. pwd où je suis. cd change dossier de travail. cd ../../ ... remonte d'un dossier de travail de .. fois. ls liste de ce contient le dossier. -a tout voir même les cachés -l détaillé -t chrologiquement man #fonction detaille la #fonction. grep recherche -i peu importe maj ou minuscule. -R avec l'info fichier - ligne. -Rl juste le fichier. ctrl+l efface les commandes passées.
mkdir #NOM crée le dossier #nom dans le dossier de travail. touch #fichier crée le #fichier. cp copie mv déplace rv supprime rv -r supprime tout ds le dossier même les sous dossiers. echo #fichier redirigé vers un #fichier. cat #fichier sort le contenu de #fichier pour le mettre ds la console. cat #fichier > change l'interieur cat # >> copie à la suite. sort tri alphabetique. uniq retire les doublons. sed '#action/#mot1/#mot2' fait l'#action sur #mot1 par #mot2.
#logiciel -v ou --version indique la version presente du #logiciel sudo apt.get install #appli installe l'#appli. sudo apt.get remove #appli supprime l'#appli. sudo apt.get purge #appli supprime l'#appli et tous ce qui lui est attaché.
