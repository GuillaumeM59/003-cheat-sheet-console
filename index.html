Navigons!

  • tabcomplémente la synthaxe.
  • pwdoù je suis.
  • cdchange dossier de travail.
  • cd ../../ ...remonte d'un dossier de travail de ../ fois.
  • lsliste de ce contient le dossier.
  • -atout voir même les cachés.
  • -ldétaillée
  • -tchronologiquement
  • man #fonctiondétaille la #fonction.
  • greprecherche
  • -ipeu importe maj ou minuscule.
  • -Ravec l'info fichier - ligne.
  • -Rljuste le fichier.
  • ctrl+lefface les commandes passées.

Action!

  • mkdir #nomcrée le dossier #nom dans le dossier de travail.
  • touch #fichiercrée le #fichier.
  • cpcopie
  • mvdéplace
  • rvsupprime
  • rv -rsupprime tout dans le dossier même les sous-dossiers.
  • echo #fichierredirigé vers un #fichier.
  • cat #fichiersort le contenu de #fichier pour le mettre ds la console.
  • cat #fichier >change l'interieur
  • cat # >>copie à la suite.
  • sorttri alphabetique.
  • uniqretire les doublons.
  • sed '#action/#mot1/#mot2' fait l'#action sur #mot1 par #mot2.

Petit Plus...

  • #logiciel -v ou --versionindique la version presente du #logiciel
  • sudo apt.get install #appli installe l'#appli.
  • sudo apt.get remove #appli supprime l'#appli.
  • sudo apt.get purge #appli supprime l'#appli et tous ce qui lui est attaché.
